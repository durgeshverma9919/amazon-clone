# Amazon Clone

A responsive Amazon homepage clone built using **HTML and CSS**.
This project focuses on layout design, responsiveness, and real-world UI structure.

## 🚀 Features

* Amazon-style navigation bar
* Product sections with cards
* Responsive layout for mobile and desktop
* Clean UI using Flexbox and media queries

## 🛠 Tech Stack

* HTML5
* CSS3
